# Algoritmos

## Problemas

- El problema del agente viajero: Dada una *gráfica* completa $G$ donde todas las aristas tienen un peso asignado, determinar un ciclo que recorra todos los vértices y regrese al punto de partida y que tenga consto mínimo.
- El problema del emparejamiento perfecto en gráficas bipartitas
- El problema del **número cromático** $\chi(G)$ de una gráfica $G$.
- El problema de determinar si una gráfica tiene un ciclo hamiltoniano.
