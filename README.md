# Algoritmos

## Problemas

- El problema del agente viajero: Dada una *gráfica* completa $G$ donde todas las aristas tienen un peso asignado, determinar un ciclo que recorra todos los vértices y regrese al punto de partida y que tenga consto mínimo.
- El problema del emparejamiento perfecto en gráficas bipartitas
- El problema del **número cromático** $\chi(G)$ de una gráfica $G$.
- El problema de determinar si una gráfica tiene un ciclo hamiltoniano.
- El problema del **flujo máximo**: Dada una red de flujo con capacidades en las aristas, determinar el flujo máximo que puede enviarse desde un vértice fuente $s$ a un vértice sumidero $t$.
- El problema del **árbol generador mínimo**: Dada una gráfica conexa $G$ con pesos en las aristas, encontrar un subconjunto de aristas que conecte todos los vértices con costo total mínimo.
- El problema de **cobertura de vértices**: Dado un grafo $G = (V,E)$, encontrar el conjunto de vértices de cardinalidad mínima tal que cada arista tenga al menos uno de sus extremos en dicho conjunto.
- El problema de **camino más corto**: Dada una gráfica $G$ con pesos en las aristas y dos vértices $u$ y $v$, encontrar el camino de $u$ a $v$ con costo mínimo.
- El problema de la **clique máxima**: Dado un grafo $G$, encontrar el subgrafo completo de cardinalidad máxima.
- El problema de **isomorfismo de grafos**: Dados dos grafos $G$ y $H$, determinar si son isomorfos.
